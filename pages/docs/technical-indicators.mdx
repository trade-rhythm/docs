# Technical Indicators

We provide a number of technical indicators that you may incorporate into your strategies.

> Our technical indicators are [open soure](https://github.com/trade-rhythm/technical-indicators) and you're welcome to view and contribute to the growing list of items.

## Example Usage

```js
import { EMA, Cross, parse } from "https://traderhythm.io/std/v1/mod.js";

// init some indicators
const ema = new EMA(9);
const cross = new Cross();

// do something with them
function handleNewData(bar) {j
  const value = ema.next(bar.close); // or ema.nextBar(bar);
  if (cross.over(value, bar.close)) {
    // Do something!
  }
}

// serialize indicators to JSON
const state = { ema, cross };
const json = JSON.stringify(state);

// deserialize
const newState = parse(json);
```
